# GradPhotoStitcher
### Stitches together the preview images from flashphotography and generates an image

<br></br>
## Requirements
```py
pip install pillow
pip install requests
```
Follow the steps in main.py to configure the script to fetch all photo metadata you want to export

<br></br>
## Running
```
python main.py
```
